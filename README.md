# Killer-master
基于Python的窗口关键词检测关闭工具
### 配置文件
```bash
config.json
{
    "gametag" : ["快游", "游戏", "模拟器","云·"],  //窗口标题关键词
    "bilibili_tag" : ["王者荣耀","原神"],         //b站标签关键词
    "timewait" : 30,                             //检索间隔时间
    "deltatime" : 100                            //edge浏览器历史记录范围
} 
```
